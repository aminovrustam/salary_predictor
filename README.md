# Прогнозирование зарплаты по тексту вакансии

Модель предсказывает `log_salary_from` на основе текстовых полей вакансии.  

## Этапы

- **Предобработка**  
  - Объединение `location` + `company` → `complocation`  
  - Добавление склонения опыта в `title` (c опытом X лет/года/год)  
  - Лемматизация, удаление стоп‑слов, токенизация

- **Признаки** – 4 текстовых поля: `title`, `complocation`, `skills`, `description`

- **Архитектура**  
  4 независимые Bi-LSTM (по одной на каждое поле) → конкатенация →  
  self‑attention + cross‑attention → финальный регрессор (MLP)

- **Результат** – R² = 0.6592 на валидационной выборке
